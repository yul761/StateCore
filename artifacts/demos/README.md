# Demo Artifacts

This directory holds curated, stable evidence artifacts that are intended for repository browsing.

Unlike `benchmark-results/`, these files are not meant to capture every local run.
They are selected samples that make the project's claims easier to evaluate quickly.

Current artifacts:

- `visible-comparison-latest.md`
- `visible-comparison-latest.json`

The visible comparison artifact shows the same event sequence evaluated two ways:

- Project Memory
- a direct-model rolling-summary baseline

The goal is to make drift reduction observable in plain language rather than only in aggregate metrics.
